INSTALLS
cd stadvdb-web
npm install

cd stadvdb-web/backend
npm install

Start All Backend Servers
From the stadvdb-web/backend folder, run:
	npm start

This will launch three backend servers:
- http://ccscloud.dlsu.edu.ph:60751 (main)
- http://ccscloud.dlsu.edu.ph:60752 (A)
- http://ccscloud.dlsu.edu.ph:60753 (B)

Frontend Env File
In stadvdb-web/ create .env file:
	REACT_APP_API_URL=http://localhost:60751
- Change to 60752 or 60753

To start it, start in the backend then the root folder of the web
/stadvdb-web/backend
npm start

/stadvdb-web
npm start
