INSTALLS
cd stadvdb-web
npm install

cd stadvdb-web/backend
npm install

Start All Backend Servers
From the stadvdb-web/backend folder, run:
	npm start

This will launch three backend servers:
- http://localhost:5000 (main)
- http://localhost:5001 (A)
- http://localhost:5002 (B)

Frontend Env File
In stadvdb-web/ create .env file:
	REACT_APP_API_URL=http://localhost:5000
- Change to 5001 or 5002

To start it, start in the backend then the root folder of the web
/stadvdb-web/backend
npm start

/stadvdb-web
npm start
